REAL-TIME-ACCENT-TRANSLATION

A Flask + Python based project that performs accent detection, accent translation, and text-to-speech in a single pipeline. The system takes voice input, identifies the speaker’s accent, converts it into a user-selected target accent, and outputs the translated audio.

Features
----------
*Speech-to-Text Conversion
*Accent Detection
*Accent Translation (Source → Target Accent)
*Text-to-Speech Output
*User-Friendly Web Interface (Flask + HTML/CSS)

Tech Stack
-----------
Python
Flask (Backend Web Framework)
Machine Learning / Deep Learning Models
Speech Processing Libraries
HTML / CSS / JavaScript (Frontend)

The goal of this project is to create a smooth and realistic conversion of spoken audio from one accent to another, enabling easier communication and understanding across diverse accents.
